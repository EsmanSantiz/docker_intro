# Introducción de Docker: Reto del Mike

Este repositorio contiene la evidencia técnica y el desarrollo del primer reto de la materia **Cómputo en la Nube**.

## Objetivo del Proyecto
El propósito de esta práctica es configurar un entorno de desarrollo aislado utilizando **contenedores**, desplegando un nodo de **Ubuntu** para la integración de herramientas de código abierto.

## Tecnologías y Herramientas
* **Docker Desktop**: Motor de contenedores para Windows.
* **Ubuntu Linux**: Sistema operativo base dentro del contenedor.
* **Git**: Herramienta para la clonación del proyecto **Infoga**.
* **GitHub Desktop**: Gestión del repositorio y control de versiones.

## Desarrollo Técnico Realizado
1. **Verificación**: Comprobación de la versión de Docker instalada en el sistema anfitrión.
2. **Despliegue**: Creación del contenedor interactivo `nodo_ubuntu`.
3. **Instalación**: Configuración interna del nodo con Git y dependencias necesarias.
4. **Persistencia**: Validación de comandos `start` y `stop` para asegurar la disponibilidad de los archivos.
